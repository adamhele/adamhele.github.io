Installation guide

WolfLoom is intentionally simple. Even if you are not very technical, you can follow these steps. Choose your operating system below and follow the instructions carefully.

Install WolfLoom on Windows

  1. Install WolfLoom
    Download the latest Windows version from here to your device.
  2. Run WolfLoom
    Find the app in File Explorer and double click it to run. It may trigger a false positive on your antivirus so click accept.
  3. Visit WolfLoom in your browser
    Open your web browser (Edge, Chrome, Firefox, etc.) and go to: http://0.0.0.0:5000/
    You should now see the WolfLoom home server interface.

Install WolfLoom on Linux

  1. Install WolfLoom
    Download the latest Linux version from here to your device.
  2. Give WolfLoom permissions
    Run WolfLoom through command-line by typing
    tar -xzvf wolfloom-server-linux.tar.gz
chmod +x wolfloom-server-linux
./wolfloom-server-linux
  3. Visit WolfLoom in your browser
    Open your browser and go to: http://0.0.0.0:5000/

Run WolfLoom on Android (via Python)

There is no dedicated Android app yet, but you can still run WolfLoom on Android by running the Python code directly.

  1. Install a Python environment on Android
    Install a Python app such as Termux or another Python environment from the Play Store or your preferred source.
  2. Install Python 3
    Inside your chosen environment, install Python 3 (for example, in Termux you might run: pkg install python).
  3. Install pip
    Pip is usually included with Python. To check, run: pip --version.
  4. Install Flask and psutil
    Run: pip install flask psutil
  5. Download the WolfLoom code
    Use the Android browser to download the project ZIP from: GitHub – wolfloom/wolfloom.github.io
    Extract the ZIP and locate the server folder.
  6. Run the server
    In your Python environment, navigate to the server folder and run: python host.py
    Keep this running.
  7. Visit WolfLoom in your browser
    On the same Android device, open your browser and go to: http://0.0.0.0:5000/

Frequently asked questions

Do I need to be technical to use WolfLoom?

No. WolfLoom is specifically designed for people who are not very tech savvy, as well as those who are tech savvy but just want a quick server to set up. If you wanted, you could add more features to your own private version by downloading and editing the python code to make custom features just for you! The instructions are step-by-step and written in everyday language to be as accessible as possible.

Do I need a fast computer to run Wolfloom

No. WolfLoom is meant to be as easy to run as possible. This means you can run it on old devices you have lying around your house.

Is WolfLoom really free?

Yes. WolfLoom is free and open source. You can use it, share it, and even modify it as long as you follow the license terms in the LICENSE.

What happens if I close the WolfLoom window or terminal?

WolfLoom runs as long as the program or terminal window is open. If you close it, the server stops. To use it again, just run the program or command again. Please don't close the window or terminal window to stop it however, as this may break it, press CTRL + C in the terminal or stop server in the app, to safely end to program before closing it.

Can I access WolfLoom from other devices on my network?

Yes, you can. Instead of 0.0.0.0, you would use the local IP address of the device running WolfLoom. For example, http://192.168.1.10:5000/. This depends on how you configured it in the app.
